AI-Powered Inventory Management Platform built with Streamlit, SQLite, SQLAlchemy, Plotly and Gemini AI.

## Features

### 📦 Inventory Management
- Product Catalog
- Stock Tracking
- Inventory Valuation
- Stock Adjustments
- Reorder Levels

### 🚚 Supplier Management
- Supplier Directory
- Contact Information
- Lead Time Tracking

### 📄 Purchase Orders
- Create Purchase Orders
- PO Status Tracking
- PDF Export
- Direct Supplier Communication

### 📊 Analytics
- Inventory Value Analysis
- Category Analysis
- ABC Classification
- Fast/Slow Moving Products
- Stockout Forecasting

### 🤖 AI Copilot
- Gemini AI Integration
- Inventory Health Summary
- Reorder Recommendations
- Smart Purchase Orders
- Conversational Chat History

### 🔐 Security
- Login System
- Role-Based Access Control
- Admin / Manager Permissions

## Tech Stack

- Python
- Streamlit
- SQLite
- SQLAlchemy
- Pandas
- Plotly
- Google Gemini AI

## Installation

```bash
git clonehttps://github.com/aniketjain2023a-ops/InventoryIQ.git
cd InventoryIQ

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

streamlit run app.py
```
