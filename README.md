# 🚀 InventoryIQ

AI-Powered Inventory Management Platform built with Streamlit, SQLite, SQLAlchemy, Plotly and Gemini AI.

## Features

### 📦 Inventory Management
- Product Catalog
- Stock Tracking
- Inventory Valuation
- Stock Adjustments
- Reorder Levels

### 🚚 Supplier Management
- Supplier Directory
- Contact Information
- Lead Time Tracking

### 📄 Purchase Orders
- Create Purchase Orders
- PO Status Tracking
- PDF Export
- Direct Supplier Communication

### 📊 Analytics
- Inventory Value Analysis
- Category Analysis
- ABC Classification
- Fast/Slow Moving Products
- Stockout Forecasting

### 🤖 AI Copilot
- Gemini AI Integration
- Inventory Healt
